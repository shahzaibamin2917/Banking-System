Banking-System
This project is a simple **Banking System** developed in **C++** using the principles of **Object-Oriented Programming (OOP)**. It is designed to simulate basic banking operations while demonstrating the use of classes, objects, constructors, and vectors in C++.

The system includes three main classes: **Customer**, **Account**, and **Transaction**. Customers can create and manage bank accounts, perform deposits and withdrawals, transfer funds between accounts, and view their account information. Every banking operation is automatically recorded in the transaction history, allowing users to review recent transactions at any time.

The project also includes balance validation to prevent withdrawals or transfers when there are insufficient funds. A menu-driven interface makes the application easy to use and suitable for beginners who are learning C++ programming and object-oriented concepts.
Features
* Create and manage customer accounts
* Deposit money into an account
* Withdraw money with balance checking
* Transfer funds between accounts
* Store and display transaction history
* View account details and current balance
* Menu-driven console interface
* Beginner-friendly and easy-to-understand C++ code

This project is ideal for students who want to strengthen their understanding of Object-Oriented Programming and implement a real-world banking application using C++.
